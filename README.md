# xivdiagrams

A web app for creating FFXIV raid diagrams, heavily inspired by tools like [Excalidraw](https://excalidraw.com/), Fabric.js, Figma, and others.

Feature roadmap:

- fight-specific backgrounds
- searchable debuffs for players
- grid, snap-to
